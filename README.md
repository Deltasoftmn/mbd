# MBD Tax Tactic - Tax Advisory Services

A professional landing page for MBD TAX TACTIC LLC, a Mongolian tax advisory and accounting services company.

## Tech Stack

- **React** - UI library
- **Next.js** - React framework
- **Tailwind CSS** - Styling

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
